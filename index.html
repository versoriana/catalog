About the Bibliotheca Manuscripta Versoriana

A digital catalog of manuscripts related to Johannes Versoris and his works.

Manuscripts

Our manuscript collection contains detailed descriptions of codices containing works by or related to Johannes Versor. Each manuscript entry includes codicological information, content descriptions, and scholarly notes.

Browse Manuscripts

Persons

The person register documents individuals associated with the manuscript tradition, including scribes, owners, and other historical figures connected to the preservation and transmission of Versor's works.

Browse Persons

Features

  • Cross-references: Links between persons and manuscripts they are associated with
  • TEI encoding: All data is encoded following TEI P5 guidelines for scholarly digital editions
  • CETEIcean integration: Native display of TEI XML in the browser

Technical Information

This digital edition is built using:

  • TEI P5: Text Encoding Initiative guidelines for scholarly digital editions
  • CETEIcean: JavaScript library for displaying TEI XML in web browsers
  • Bootstrap 5: CSS framework for responsive design
